## What is AI Babysitter?

AI Babysitter is an intelligent virtual assistant designed to engage, educate, and keep children safe through:

- **Interactive Conversations** powered by NVIDIA Nemotron AI
- **Educational Activities** including story time, homework help, and games
- **Safety Monitoring** with real-time parent alerts
- **Multi-Child Support** with individual profiles and settings
- **RAG-Powered Responses** using trusted sources (CDC, CPSC, NIH)
